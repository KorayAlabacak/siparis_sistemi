Bu proje sipariş sistemidir 3 girişten oluşur admin,kargocu ve müşteri diye.
Dosyalarda iki tane pyton dosyası vardır.
Ham dosyası benim kendimin yapay zekasız yazdığım olan database kaydetmeyen koddur.
Diğer ana kod ise yapay zekadan yardım alınarak database kaydedip admin ve kargocu girişlerininin şifrelerini gizleyen destekli bir projedir.
Bu projedeki amacım database e veri kaydetmesini daha iyi kavrayıp şifreleme sistemini öğrenmek.
Prjoede kullandıklarım sözlük, fonksiyonlar, database ve listeleme.
Projemedeki fonksiyonlar parametresiz ve geriye değer döndürmeyen fonksiyonlardır.
